# Sandbox

Zona de proyectos en desarrollo o experimentales. Cada subcarpeta representa un proyecto independiente con la siguiente estructura sugerida:
- `docs/`: diseño y documentación operativa.
- `csv/`: inventarios y tableros.
- `src/`: código fuente del artefacto principal.
- `tests/`: pruebas automatizadas.
- Artefacto principal (`*.csproj` o `*.ps1`) en la raíz del proyecto.

Consulta el README y AGENTS del proyecto antes de trabajar.
