# ProyectoEjemplo

Plantilla de proyecto Sandbox para aplicaciones CLI. Incluye código .NET multi-target y pruebas xUnit.

## Estructura
- `docs/`: notas de diseño y bitácoras.
- `csv/`: inventarios de artefactos.
- `src/`: código fuente principal.
- `tests/`: pruebas automatizadas.
- `ProyectoEjemplo.csproj`: artefacto principal (aplicación de consola).

## Uso
- Restaurar y compilar: `dotnet build ProyectoEjemplo.csproj`
- Probar: `dotnet test tests/ProyectoEjemplo.Tests.csproj`
