# AGENTS – ProyectoEjemplo

- Sigue la política global y el AGENTS raíz antes de modificar este proyecto.
- Prefiere implementar lógica en C# multi-target (`net8.0;net7.0;net6.0`); si necesitas scripts, usa PowerShell compatible con PS 5.1 y 7+.
- Mantén documentación en `docs/` y actualiza los inventarios en `csv/` cuando cambie la arquitectura.
- Ejecuta `dotnet test` para validar cambios en código C# y PSScriptAnalyzer para scripts PowerShell.
