# Preferencias del usuario

- Idioma: español, tono técnico y conciso.
- Entregas: primero el resultado accionable (código o scripts listos), después una explicación breve (máximo 1–3 párrafos).
- Herramientas preferidas: PowerShell y C#; evitar Python salvo petición explícita.
- Entorno: desarrollo en Windows 10/11 con PowerShell 7.x, procurando compatibilidad con Windows PowerShell 5.1 cuando se indiquen scripts.
